# LinkedIn Lead CRM System

Simple CRM system to manage LinkedIn leads, outreach, and follow-ups using Google Sheets and Apps Script.

---

## Overview

This project helps manage B2B leads from LinkedIn in a structured way.

Instead of tracking conversations manually, this system allows you to:

* Store leads
* Track outreach status
* Manage follow-ups
* Organize conversations

---

## Key Features

* Centralized lead tracking
* Outreach status management
* Follow-up tracking
* Notes for conversations
* Lightweight CRM system

---

## Data Structure

The system stores:

* Name
* LinkedIn Profile URL
* Company
* Position
* Status (New / Messaged / Replied / Closed)
* Last Contacted Date
* Notes

---

## How It Works

1. Add LinkedIn leads manually into Google Sheets
2. Update status after outreach
3. Track follow-ups
4. Manage pipeline in one place

---

## Setup Instructions

### Step 1: Create Google Sheet

Create a sheet named:
Leads

---

### Step 2: Add Columns

Name
LinkedIn URL
Company
Position
Status
Last Contacted
Notes

---

### Step 3: Add Script

Go to:
Extensions → Apps Script

Paste script from:
/scripts/linkedin-crm.js

---

### Step 4: Run Script

Use functions to update status and track follow-ups.
